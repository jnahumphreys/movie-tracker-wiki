# User Journeys

Expected step-by-step behaviour for each core flow. These are the behavioural source of truth — Design covers how each step looks, Engineering covers how it's implemented.

# Journeys

* ✅ [App First Launch](/product/user-journeys/app-first-launch.md) - Expected behaviour for a user's first launch of the app.
* ✅ [Searching the OMDb to Add a Movie](/product/user-journeys/searching-omdb.md) - Expected behaviour for searching and adding movies via the OMDb API.
* ✅ [Searching the User's Library](/product/user-journeys/searching-library.md) - Expected behaviour for filtering the to-watch and watched library views.
* ✅ [Managing the Local Library](/product/user-journeys/managing-local-library.md) - Expected behaviour for editing or removing movies already in the library.
* ✅ [Data Persistence Errors](/product/user-journeys/data-persistence-errors.md) - Expected behaviour when reading from or writing to local storage fails.
